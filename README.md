# foundations-of-robotics

Course materials for the ***Foundations of Robotics*** course (*École des Mines de Nancy*, 2026).

This repository contains lecture slides, code examples, and lab material used throughout the course.

## Repository structure

- [`lectures`](lectures/README.md): lecture slides and example code
- [`lab-material`](lab-material/README.md): practical lab modules, simulations, and supporting material

## Getting started

- For lectures and example code, follow the instructions in [`lectures/README.md`](lectures/README.md).
- For hands-on labs, start from [`lab-material/README.md`](lab-material/README.md) and refer to **Lab-notes.pdf**.
